# ASCII Value Finder

## Description
A Java application to find and display the ASCII value of a character input by the user.

## Features
- Console and GUI interface options
- Input validation
- Modular design

## Usage
### Console Mode
```bash
java AsciiApp
```

### GUI Mode
Set `useGUI = true` in `AsciiApp.java` and run again.

## Requirements
- Java 8+
